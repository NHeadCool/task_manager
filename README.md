Инструкция по запуску:

1. Перейдите в Pycharm
2. Клонируйте репо:
git clone https://dbdiagram.io/d/694aab96dbf05578e6633143
3. Включите добавьте виртуальное окружение 
python -m venv env
.\venv\Scripts\Active.ps1
4. Перейдитсе в cd task_manager
5. Установите зависимости
pip install -r requirements.txt
6. Перейдите выше
cd ..
7. Запустите сервис
python -m task_manager.app
