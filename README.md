# task_manager

Сервис разрабатывается в рамках модульного задания
Общее видение сервиса:
task_manager — это помощник для учебных и проектных групп, который помогает:
    организовывать совместную работу,
    не терять задачи и дедлайны,
    координировать людей, состоящих сразу в нескольких группах,
    снижать когнитивную нагрузку.

Особенность: Пользователь участвует в нескольких независимых группах, и сервис помогает ему управлять этой сложностью.

Проблемы пользователя:
    Задачи разбросаны по чатам,
    Дедлайны забываются,
    Непонятно, кто за что отвечает,
    Один человек участвует в нескольких проектах = путаница,
    Нет общей картины загрузки.



Инструкция по запуску:
1. Перейдите в pycharm
2. Клонируйте репо:
git clone https://dbdiagram.io/d/694aab96dbf05578e6633143
3. Включите добавьте виртуальное окружение (перейдите в cd task_manager)
python -m venv venv
.\venv\Scripts\Activate.ps1
5. Установите зависимости
pip install -r requirements.txt
6. Перейдите выше
cd ..
7. Запустите сервис
python -m task_manager.app
